## 📖 О проекте

**DiscountCalculator** — это веб-приложение, которое позволяет рассчитать итоговую стоимость товара с учётом скидки. Пользователь вводит исходную цену и указывает статус VIP-пользователя, а программа мгновенно показывает:

- Итоговую цену после скидки

Проект создан для отработки навыков работы с событиями JavaScript, обработкой пользовательского ввода и стильным оформлением интерфейса.
