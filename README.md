## 📖 О проекте

**DiscountCalculator** — это веб-приложение, которое позволяет рассчитать итоговую стоимость товара с учётом скидки и представить ее в 3 разных валютах. Пользователь вводит исходную цену и указывает статус VIP-пользователя, а программа мгновенно показывает:

- Итоговую цену после скидки

Проект создан для отработки навыков работы с событиями JavaScript, обработкой пользовательского ввода и стильным оформлением интерфейса.

---

## 🎯 Функционал

| Функция | Описание |
|---------|----------|
| Расчёт скидки | Ввод цены и VIP-статуса → мгновенный расчёт |
| Конвертация | Конвертация в RUB, USD, EUR |
| Форматирование | Результат отображается с двумя знаками после запятой |
| Валидация | Обработка отрицательных чисел и некорректного ввода |
| Современный UI | Адаптивный, современный дизайн |
